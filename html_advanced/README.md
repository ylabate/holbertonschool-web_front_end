# html advanced

Ce dossier contient la structure HTML avancée pour le site web Techium.

- `about.html`: La page "À propos" (About) du site Techium.
- `article.html`: Un modèle de page d'article pour le site Techium.
- `contact.html`: La page de contact du site Techium.
- `index.html`: La page d'accueil (Homepage) du site Techium.
- `latest_news.html`: La page des dernières actualités (Latest news) du site Techium.
- `styleguide.html`: Le guide de style (Styleguide) du site Techium.
